# Auto Accept Agent

[English](README_EN.md) | **中文**

> 一个 VS Code / Antigravity / Cursor 扩展，自动接受 AI 代理的操作请求，让你从频繁的手动审批中解放出来。

---

## ✨ 核心功能

- **自动接受操作** — 支持自动接受文件编辑、终端命令、重试提示等 14 种操作类型
- **可配置操作类型** — 通过设置面板勾选需要自动接受的操作，精确控制自动化范围
- **后台模式** — 同时管理多个 AI 对话标签页，无需手动切换
- **危险命令拦截** — 内置保护机制，自动阻止 `rm -rf /` 等破坏性命令
- **实时状态指示** — 状态栏颜色标识当前状态（紫色=运行中，绿色=已完成）
- **智能端口检测** — 自动扫描 CDP 调试端口，无需手动配置
- **多语言支持** — 自动检测 IDE 语言，支持英文和简体中文

## 📦 安装

1. 下载最新的 `.vsix` 文件
2. 打开 VS Code / Antigravity / Cursor
3. `Ctrl+Shift+P` → 输入 `Install from VSIX`
4. 选择 `.vsix` 文件并重启 IDE

## 🚀 使用方法

1. 点击状态栏 `Auto Accept: OFF` 启用
2. 首次使用需允许快捷方式更新（添加 `--remote-debugging-port=9000`）
3. 启用后自动运行，状态栏显示 `Auto Accept: ON`

## ⚙️ 配置项

| 配置 | 说明 | 默认值 |
|------|------|--------|
| CDP 端口 | 调试协议端口号 | 自动检测 |
| 自动接受文件编辑 | 是否自动应用文件更改 | 开启 |
| 语言覆盖 | 强制指定界面语言 | 自动 |
| Overlay 模式 | 后台模式显示方式 | 关闭 |
| 操作类型选择 | 勾选需要自动接受的操作 | 仅 run |
